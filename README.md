# SQL para Análise de Dados

Este repositório foi criado com o objetivo de fortalecer meus conhecimentos em SQL e consolidar habilidades essenciais para a área de Análise de Dados. Aqui reúno exercícios, consultas e desafios práticos desenvolvidos durante minha jornada de aprendizado, buscando aplicar conceitos utilizados em cenários reais de manipulação, transformação e análise de dados.

# Objetivos
- Aprimorar a escrita de consultas SQL eficientes.
- Desenvolver habilidades de extração, tratamento e análise de dados.
- Praticar conceitos fundamentais e avançados da linguagem.
- Construir uma base sólida para apoiar atividades de Análise de Dados e Business Intelligence.
- Conteúdo
- Consultas básicas e avançadas
- Filtros e ordenações
- Funções de agregação
- Joins
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Manipulação e transformação de dados
- Exercícios e estudos de caso

# Competências Desenvolvidas

- Extração e consulta de dados
- Limpeza e transformação de dados
- Análise exploratória
- Modelagem relacional
- Otimização de consultas
- Resolução de problemas com SQL
- Pensamento analítico

# Tecnologias
- SQL

# Aprendizado Contínuo
Este repositório é atualizado continuamente à medida que novos conceitos são estudados e aplicados, refletindo minha evolução técnica e meu compromisso com o desenvolvimento profissional na área de dados.
